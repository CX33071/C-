范围for循环

语法：`for(类型 变量名：数组名/容器)`

关键变体：

| 写法                                 | 含义                      | 适用场景                 |
| ------------------------------------ | ------------------------- | ------------------------ |
| `for (auto elem : container)`        | 拷贝元素（值传递）        | 仅读取元素，且元素体积小 |
| `for (auto& elem : container)`       | 引用元素（避免拷贝）      | 修改元素、元素体积大     |
| `for (const auto& elem : container)` | 常量引用（只读 + 无拷贝） | 仅读取元素，优先推荐     |
