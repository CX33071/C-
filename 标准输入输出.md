标准输入输出

1.`C++`的标准输入输出通过`<iostream>`头文件实现，无需.h后缀，需要启用命名空间：`using namespace std`,不写命名空间就要用`std::cout`全称

全局启用：

```c
#include <iostream>
using namespace std; // 简化写法，新手推荐

int main() {
    // 输入输出代码写在这里
    return 0;
}

```

局部指定：

```c
#include <iostream>

int main() {
    std::cout << "Hello" << endl; // 每次加 std::
    return 0;
}

```

2.标准输出:`cout`

(1).搭配<<**插入运算符**使用，支持链式调用

```c
#include <iostream>
using namespace std;

int main() {
    // 输出字符串常量
    cout << "Hello, C++!" << endl; // endl：换行+刷新缓冲区（等价于 C 的 \n + fflush(stdout)）
    
    // 输出变量（无需格式控制符，自动匹配类型）
    int a = 10;
    double b = 3.14;
    char c = 'A';
    cout << "a = " << a << ", b = " << b << ", c = " << c << "\n"; // \n 仅换行，不刷新
    
    return 0;
}

```

(2).输出格式控制

用`<iomanip`头文件的控制符调整格式

```c
#include <iostream>
#include <iomanip> // 格式控制头文件
using namespace std;

int main() {
    double pi = 3.1415926535;
    
    // 1. 控制浮点数精度（保留 2 位小数）
    cout << "pi（保留2位）：" << fixed << setprecision(2) << pi << endl;
    
    // 2. 控制输出宽度（不足补空格，默认右对齐）
    cout << "宽度10：" << setw(10) << 123 << endl;
    
    // 3. 左对齐（配合 setw）
    cout << "左对齐10：" << left << setw(10) << 123 << "结束" << endl;
    
    // 4. 进制转换（十进制/八进制/十六进制）
    int num = 255;
    cout << "十进制：" << dec << num << endl;  // 默认十进制
    cout << "八进制：" << oct << num << endl;  // 八进制（无前导 0）
    cout << "十六进制：" << hex << num << endl;// 十六进制（小写，无前导 0x）
    cout << "十六进制大写：" << uppercase << hex << num << endl; // 大写
    
    return 0;
}

```

(3).错误输出：`cerr/clog`

`cerr`:标准错误输出，不缓冲，立即输出

`clog`:标准日志输出，缓冲，批量输出

```c
cout << "普通输出（可能延迟）" << endl;
cerr << "错误信息（立即显示）" << endl;
clog << "日志信息（缓冲后显示）" << endl;

```

3.标准输入:`cin`

自动忽略空格/换行/制表符，搭配**提取运算符**>>使用

(1)读取单个变量

```c
#include <iostream>
using namespace std;

int main() {
    int a;
    double b;
    char c;
    string s; // 字符串（需包含 <string> 头文件）
    
    cout << "请输入一个整数、一个小数、一个字符、一个字符串：" << endl;
    cin >> a >> b >> c >> s; // 链式读取，空格分隔输入
    
    cout << "你输入的是：" << endl;
    cout << "a = " << a << ", b = " << b << ", c = " << c << ", s = " << s << endl;
    
    return 0;
}

```

(2)读取带空格的字符串

`cin >> s` 会以空格 / 换行作为分隔符，无法读取带空格的字符串，此时用 `cin.getline()`：

```c
#include <iostream>
#include <string> // string 头文件
using namespace std;

int main() {
    char str[100]; // 字符数组（指定最大长度）
    string s;      // string 类型（更推荐，无需手动指定长度）
    
    cout << "请输入带空格的字符串（字符数组）：" << endl;
    cin.getline(str, 100); // 读取到换行符结束，最多读 99 个字符（留 1 个存 '\0'）
    
    cout << "请输入带空格的字符串（string 类型）：" << endl;
    getline(cin, s); // string 专用的 getline（需包含 <string>）
    
    cout << "字符数组：" << str << endl;
    cout << "string：" << s << endl;
    
    return 0;
}

```

(3)读取单个字符

`cin >> c` 会忽略空格 / 换行，若要读取包括空格、换行在内的单个字符，用 `cin.get()`：

```c
#include <iostream>
using namespace std;

int main() {
    char c1, c2;
    cout << "请输入两个字符（含空格）：" << endl;
    c1 = cin.get(); // 读取第一个字符（包括空格/换行）
    c2 = cin.get(); // 读取第二个字符
    
    cout << "c1 = '" << c1 << "', c2 = '" << c2 << "'" << endl;
    return 0;
}

```

4.区别

| 功能             | C                    | C++                    | 优势             |
| ---------------- | -------------------- | ---------------------- | ---------------- |
| 输入             | `scanf("%d",&a);`    | `cin>>a`               | 无需&直接去地址  |
| 输出             | `printf("%d",a);`    | `cout<<a`              | 无需%d等格式符   |
| 字符串输出       | `printf("%s",s);`    | `cout<<s`              | 支持s直接输出    |
| 读取代空格字符串 | `fgets(str,n,stdin)` | `getline(cin,s)`       | 无需手动管理长度 |
| 换行             | `printf("\n")`       | `cout<<endl或cout<<\n` | `endl`刷新缓冲区 |

5.`cin` 读取时会忽略空白字符（空格、制表符、换行），如果需要读取这些字符，用 `cin.get()` 或 `getline`。

混用 `cin >>` 和 `getline` 可能出现问题（`cin >>` 会留下换行符在缓冲区，被 `getline` 读取），解决方法：在 `getline` 前用 `cin.ignore()` 清空缓冲区：

```c
int a;
cin >> a;
cin.ignore(); // 清空缓冲区的换行符
string s;
getline(cin, s); // 正常读取带空格的字符串

```

6.统计字符串长度

```c
// 统计字符串长度：size() 和 length() 效果完全相同
    int str_length = input_str.size();  // 推荐用法（更符合 STL 容器接口规范）
     int str_length = input_str.length();  // 等价用法，兼容早期 C++ 标准
```

.是成员访问运算符，通过input_str这个字符串对象调用string类里的size()工具
