命名空间：

1.作用：解决同名变量或函数冲突的问题，类似头文件，定义函数变量,合理命名，解决冲突

2.定义命名空间：变量、函数、结构体

```c
// 定义命名空间 MyMath（存放数学相关函数）
namespace MyMath {
    const double PI = 3.1415926; // 命名空间内的常量
    double circleArea(double r) { // 命名空间内的函数
        return PI * r * r;
    }
}

// 定义另一个命名空间 MyString（存放字符串相关函数）
namespace MyString {
    void swap(char* a, char* b) { // 和其他命名空间的swap同名也没关系
        char temp = *a;
        *a = *b;
        *b = temp;
    }
}

```

3.使用命名空间里的成员

语法:**命名空间名：：成员名**（：：是作用域解析符）

```c
#include <iostream>
int main() {
    // 访问 MyMath 中的 PI 和 circleArea
    double r = 2.0;
    std::cout << "圆面积：" << MyMath::circleArea(r) << std::endl;
    std::cout << "PI：" << MyMath::PI << std::endl;

    // 访问 MyString 中的 swap
    char a = 'A', b = 'B';
    MyString::swap(&a, &b);
    std::cout << a << " " << b << std::endl; // 输出 B A

    return 0;
}

```

或使用using声明

```c
#include <iostream>
int main() {
    // using声明：指定使用 std 命名空间的 cout
    using std::cout;
    using std::endl; // 同时声明 endl
    using MyMath::circleArea; // 声明 MyMath 的 circleArea

    // 之后不用写 std:: 和 MyMath:: 了
    cout << "圆面积：" << circleArea(3.0) << endl; // 直接用 cout、circleArea
    cout << "PI：" << MyMath::PI << endl; // 没声明 PI，仍需加 MyMath::

    return 0;
}

```

4.嵌套命名空间

```c
namespace MyLib { // 外层命名空间
    namespace Math { // 嵌套命名空间（MyLib::Math）
        int add(int a, int b) { return a + b; }
    }
    namespace String { // 嵌套命名空间（MyLib::String）
        int length(const char* str) {
            int len = 0;
            while (str[len] != '\0') len++;
            return len;
        }
    }
}

// 使用嵌套命名空间的成员
int main() {
    std::cout << MyLib::Math::add(1, 2) << std::endl; // 3
    std::cout << MyLib::String::length("hello") << std::endl; // 5
    return 0;
}

```

5.命名空间别名（简称）

```c
// 原命名空间名很长
namespace MyVeryLongNamespace {
    void func() { std::cout << "hello" << std::endl; }
}

// 起别名（简称）
namespace MVL = MyVeryLongNamespace;

int main() {
    MVL::func(); // 等价于 MyVeryLongNamespace::func()
    return 0;
}

```

6.命名空间可以重复定义

```c
// 文件1：定义 MyLib 的一部分
namespace MyLib {
    int add(int a, int b) { return a + b; }
}

// 文件2：定义 MyLib 的另一部分（编译器会合并）
namespace MyLib {
    int sub(int a, int b) { return a - b; }
}

// 使用时可以访问两个函数
int main() {
    MyLib::add(3,1); // 4
    MyLib::sub(3,1); // 2
    return 0;
}

```
