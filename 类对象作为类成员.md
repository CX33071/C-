类对象作为类成员

C++类中的成员可以是另一个类的对象，我们称该成员为对象成员

```c++
class A {}
class B
{
    A a；
}
```

```c++
class A1 { public: A1() { cout << "A1 构造" << endl; } ~A1() { cout << "A1 析构" << endl; } };
class A2 { public: A2() { cout << "A2 构造" << endl; } ~A2() { cout << "A2 析构" << endl; } };
class A3 { public: A3() { cout << "A3 构造" << endl; } ~A3() { cout << "A3 析构" << endl; } };

class B {
private:
    A1 a1; // 声明顺序：A1 → A2 → A3
    A2 a2;
    A3 a3;
public:
    B() : a3(), a2(), a1() {} // 初始化列表顺序不影响构造顺序
};

int main() {
    B b;
    return 0;
}

```

**构造顺序**：对象成员（A） → 宿主对象（B）；

**析构顺序**：宿主对象（B） → 对象成员（A）；
